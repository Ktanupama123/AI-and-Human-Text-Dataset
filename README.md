# AI-and-Human-Text-Dataset
# AI vs Human Text Detection

## Overview

This project focuses on detecting whether a piece of text is written by a human or generated by artificial intelligence using Natural Language Processing (NLP) and Machine Learning techniques. The model processes text data, converts it into numerical features using TF-IDF, and classifies it using a Logistic Regression model.

## Dataset

The dataset used in this project contains text samples labeled by their author type.

**Columns in the dataset:**

* `Text` – The text content to be analyzed
* `Author` – Indicates whether the text is written by a Human or AI

The dataset used:
`data_for_preprocessing.csv`

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Natural Language Processing (NLP)
* TF-IDF Vectorization

## Project Workflow

1. Load the dataset
2. Data preprocessing
3. Remove unnecessary columns
4. Handle missing values
5. Feature extraction using TF-IDF
6. Split dataset into training and testing data
7. Train machine learning model
8. Evaluate model performance using accuracy

## Model Used

The model used in this project is:

**Logistic Regression**

It is commonly used for text classification problems and works well with TF-IDF features.
